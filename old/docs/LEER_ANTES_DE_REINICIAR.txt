PASOS PARA COMPLETAR LA INSTALACIÓN
══════════════════════════════════════════════════════════════

1. PREPARAR EL SISTEMA
   ────────────────────
   
   a) Clic derecho en: PREPARAR_PARA_REINICIO.bat
   b) Seleccionar: "Ejecutar como administrador"
   c) Seguir las instrucciones en pantalla
   d) Elegir "S" para reiniciar ahora (o reinicia manualmente después)


2. DESPUÉS DEL REINICIO
   ─────────────────────
   
   a) Windows iniciará normalmente
   b) 5 segundos después del login
   c) Se abrirá automáticamente una ventana CMD
   d) El instalador te guiará paso a paso


3. PROCESO DE INSTALACIÓN (7 PASOS)
   ─────────────────────────────────
   
   PASO 1: Verificar Python
           → Comprueba que Python esté instalado
   
   PASO 2: Instalar cloudflared
           → Descarga e instala cloudflared automáticamente
           → O te indica cómo hacerlo manualmente
   
   PASO 3: Autenticar con Cloudflare
           → Se abre tu navegador
           → Inicia sesión en Cloudflare (o crea cuenta gratis)
           → Autoriza el acceso
   
   PASO 4: Crear túnel
           → Crea el túnel "dvdcoin" automáticamente
   
   PASO 5: Configurar dominio
           → Puedes usar tu propio dominio
           → O usar un subdominio gratuito de Cloudflare
   
   PASO 6: Crear archivo de configuración
           → Genera cloudflare-config.yml automáticamente
   
   PASO 7: Instalar autostart
           → Configura inicio automático en Windows


4. DESPUÉS DE LA INSTALACIÓN
   ──────────────────────────
   
   a) El instalador preguntará si quieres iniciar el servidor
   b) Si dices "S", el servidor se iniciará automáticamente
   c) Accede a tu aplicación desde:
      • Local: http://localhost:8000
      • Público: La URL que te muestre el instalador
