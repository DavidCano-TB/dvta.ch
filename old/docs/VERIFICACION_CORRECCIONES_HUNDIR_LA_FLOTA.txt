========================================

1. ✓ PANTALLA NEGRA (BLACK SCREEN)
   - Causa: Modales con CSS conflictivo (display:none; display:flex;)
   - Solución: Eliminado display:flex; de estilos inline
   - Línea 112 (videoModal): Solo display:none; inicial
   - Línea 117 (victoryModal): Solo display:none; inicial
   - JavaScript controla display:flex cuando necesario

2. ✓ FELICIDADES APARECE AL INICIAR
   - Causa: Modal de victoria no se ocultaba al reiniciar
   - Solución: Múltiples capas de protección:
     a) window.load oculta modal y detiene confeti
     b) resetAndNewGame() oculta modal inmediatamente
     c) Bandera victoryShown previene duplicados

3. ✓ BOTONES NO FUNCIONAN DESPUÉS DE VICTORIA
   - Causa: Modal bloqueaba interacción
   - Solución: 
     a) Modal se cierra automáticamente después de 7 segundos
     b) Botones "Nueva Partida" y "Volver al Inicio" en modal
     c) resetAndNewGame() limpia todo el estado

4. ✓ JUEGO NO INICIA DESDE ESTADO 0
   - Causa: Estado previo persistía en localStorage
   - Solución:
     a) window.load limpia localStorage al cargar
     b) resetAndNewGame() limpia estado local
     c) Servidor se resetea vía API

  window.addEventListener('load', () => {
    // Ocultar modal de victoria si está visible
    const victoryModal = document.getElementById('victoryModal');
    if (victoryModal) {
      victoryModal.style.display = 'none';
    }
    
    // Detener confeti si está activo
    stopConfetti();
    
    // Limpiar localStorage relacionado con el juego
    const keysToRemove = [];
    for (let i = 0; i < localStorage.length; i++) {
      const key = localStorage.key(i);
      if (key && key.includes('hundirlaflota')) {
        keysToRemove.push(key);
      }
    }
    keysToRemove.forEach(key => localStorage.removeItem(key));
    
    console.log('🔄 Estado local limpiado - Iniciando desde 0');
  });
