🆘 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════

Problema: cloudflared not found
───────────────────────────────────────────────────────────────
Solución:
  winget install --id Cloudflare.cloudflared

O descarga desde:
  https://github.com/cloudflare/cloudflared/releases


Problema: cloudflare-config.yml not found
───────────────────────────────────────────────────────────────
Solución:
  CONFIGURAR_CLOUDFLARE.bat


Problema: El servidor no arranca
───────────────────────────────────────────────────────────────
Solución:
  1. DETENER_SERVIDOR.bat
  2. VERIFICAR_INSTALACION.bat
  3. ARRANCAR.bat


Problema: Quiero volver a ngrok temporalmente
───────────────────────────────────────────────────────────────
Solución:
  INICIAR_COMO_ADMIN_NGROK_OLD.bat

(Pero recuerda que ngrok tiene límites)


Problema: No sé mi URL de Cloudflare
───────────────────────────────────────────────────────────────
Solución:
  cloudflared tunnel info dvdcoin

O revisa:
  cloudflare-config.yml

🔄 AUTOSTART DE WINDOWS
═══════════════════════════════════════════════════════════════

Para que DVDcoin se inicie automáticamente al encender Windows:

Instalar:
  INSTALAR_AUTOSTART.bat
  (Ejecutar como Administrador)

Desinstalar:
  DESINSTALAR_AUTOSTART.bat
  (Ejecutar como Administrador)

Verificar:
  schtasks /query /tn "DVDcoin-Cloudflare"
