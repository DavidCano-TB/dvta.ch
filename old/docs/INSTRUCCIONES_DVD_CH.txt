═══════════════════════════════════════════════════════════
  📝 CONFIGURACIÓN ADICIONAL NECESARIA
═══════════════════════════════════════════════════════════

IMPORTANTE: Para que dvd.ch funcione, necesitas:

1️⃣  AÑADIR DOMINIO A CLOUDFLARE
   - Ve a: https://dash.cloudflare.com
   - Añade el dominio dvd.ch (o david.ch si es tu dominio base)
   - Sigue las instrucciones de Cloudflare

2️⃣  CAMBIAR NAMESERVERS
   - Ve al panel de tu registrador de dominios
   - Cambia los nameservers a los que Cloudflare te proporcione
   - Ejemplo:
     nameserver1.cloudflare.com
     nameserver2.cloudflare.com

3️⃣  ESPERAR PROPAGACIÓN
   - Espera 2-4 horas
   - Verifica con: nslookup dvd.ch

4️⃣  CONFIGURAR SSL (Opcional pero recomendado)
   - Cloudflare Dashboard → SSL/TLS
   - Modo: "Full"
   - Activa "Always Use HTTPS"

═══════════════════════════════════════════════════════════
  🚨 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════

PROBLEMA: "DNS_PROBE_FINISHED_NXDOMAIN"
──────────────────────────────────────
Causa: DNS no propagado aún
Solución: Espera 2-4 horas más

PROBLEMA: "502 Bad Gateway"
──────────────────────────────────────
Causa: Servidor local no está corriendo
Solución: python main.py

PROBLEMA: "ERR_SSL_VERSION_OR_CIPHER_MISMATCH"
──────────────────────────────────────
Causa: Configuración SSL incorrecta
Solución: Cloudflare Dashboard → SSL/TLS → Modo "Full"

PROBLEMA: Túnel no conecta
──────────────────────────────────────
Causa: cloudflared no está corriendo
Solución: INICIAR_DVD_CH.bat

PROBLEMA: DNS no propaga después de 24h
──────────────────────────────────────
Causa: Nameservers no cambiados correctamente
Solución: Verifica en tu registrador que los nameservers
          apuntan a Cloudflare
