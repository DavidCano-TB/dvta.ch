  ✓ GRATIS para siempre
  ✓ Mejor rendimiento
  ✓ Más seguro

ESTRUCTURA DEL SISTEMA
══════════════════════════════════════════════════════════════

ARRANCAR.bat
    ↓
INICIAR_CON_CLOUDFLARE.bat
    ↓
    ├─→ Inicia servidor Python (puerto 8000)
    │   └─→ uvicorn src.main:app --host 0.0.0.0 --port 8000
    │
    └─→ Inicia Cloudflare Tunnel
        └─→ cloudflared tunnel --config cloudflare-config.yml run
            └─→ Conecta localhost:8000 con Internet
                └─→ URL pública accesible desde cualquier lugar
