🆘 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════

Problema: El instalador no se ejecuta al reiniciar
───────────────────────────────────────────────────────────────
Solución:
  Ejecuta manualmente:
    INSTALACION_AUTOMATICA_CLOUDFLARE.bat


Problema: Python no está instalado
───────────────────────────────────────────────────────────────
Solución:
  1. El instalador te lo indicará
  2. Descarga desde: https://www.python.org/downloads/
  3. Instala Python
  4. Ejecuta el instalador nuevamente


Problema: Quiero cancelar
───────────────────────────────────────────────────────────────
Solución:
  Antes de reiniciar:
    CANCELAR_INSTALACION_AL_REINICIO.bat

  Después de reiniciar:
    Cierra la ventana del instalador


Problema: Quiero reinstalar
───────────────────────────────────────────────────────────────
Solución:
  1. Ejecuta: CONFIGURAR_INSTALACION_AL_REINICIO.bat
  2. Reinicia Windows
  3. El instalador preguntará si quieres reinstalar
