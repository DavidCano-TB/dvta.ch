  🔧 SI ALGO FALLA
═══════════════════════════════════════════════════════════════════════════

Error: "cloudflared no está instalado"
   → Descarga de: https://github.com/cloudflare/cloudflared/releases
   → Archivo: cloudflared-windows-amd64.exe
   → Renombra a: cloudflared.exe
   → Mueve a: C:\Windows\System32\

Error: "tunnel already exists"
   → No es un error, el script usará el túnel existente

Error: "unauthorized"
   → Ejecuta: cloudflared tunnel login
   → Autoriza en el navegador

Error: "connection refused"
   → Verifica que el servidor Python esté corriendo (puerto 8000)
   → Ejecuta: ARRANCAR.bat
