Para que dvd.ch funcione, necesitas configurar el DNS:

1️⃣  Añadir dvd.ch a Cloudflare Dashboard
   → https://dash.cloudflare.com
   → Add a Site → dvd.ch
   → Anota los nameservers

2️⃣  Cambiar nameservers en tu registrador
   → Ve donde compraste dvd.ch
   → Cambia a los nameservers de Cloudflare
   → Guarda los cambios

3️⃣  Esperar propagación (2-4 horas)
   → Verifica con: nslookup dvd.ch
   → Cuando veas IPs de Cloudflare, ¡está listo!
