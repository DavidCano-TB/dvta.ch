  🔧 CORRECCIONES DE SINTAXIS EN ARCHIVOS BAT
═══════════════════════════════════════════════════════════════════════════

Fecha: 27 de Mayo de 2026
Sistema: DVDBank - dvta.ch
Problema: "La syntaxe de la commande n'est pas correcte"

═══════════════════════════════════════════════════════════════════════════
  ❌ PROBLEMA IDENTIFICADO
═══════════════════════════════════════════════════════════════════════════

Los archivos BAT tenían comandos con la siguiente estructura:

    for /f "tokens=2" %%a in ('tasklist ^| findstr "python.exe"') do (
        echo PID: %%a
    )

Este patrón causa errores de sintaxis en algunos sistemas Windows,
especialmente cuando el sistema está configurado en otro idioma (francés,
español, etc.) porque:

1. El pipe (|) escapado con ^| dentro de comillas simples puede fallar
2. La redirección 2>&1 dentro del comando puede causar problemas
3. Windows CMD tiene problemas con caracteres especiales en for /f

═══════════════════════════════════════════════════════════════════════════
  ✅ SOLUCIÓN APLICADA
═══════════════════════════════════════════════════════════════════════════

Se cambió el patrón a una solución más robusta:

    tasklist | findstr "python.exe" > temp_pid.txt
    for /f "tokens=2" %%a in (temp_pid.txt) do (
        echo PID: %%a
    )
    del temp_pid.txt 2>nul

Esta solución:
✅ Evita pipes dentro de for /f
✅ Usa archivos temporales intermedios
✅ Limpia los archivos temporales después de usarlos
✅ Es compatible con todos los idiomas de Windows
✅ No tiene problemas con caracteres especiales

  🔍 DETALLES TÉCNICOS
═══════════════════════════════════════════════════════════════════════════

ANTES (PROBLEMÁTICO):
───────────────────────────────────────────────────────────────────────────
for /f "tokens=2" %%a in ('tasklist ^| findstr "python.exe"') do (
    echo PID: %%a
)

Problemas:
• El ^| puede no funcionar en todos los sistemas
• Las comillas simples pueden causar problemas
• Difícil de depurar cuando falla

DESPUÉS (CORREGIDO):
───────────────────────────────────────────────────────────────────────────
tasklist | findstr "python.exe" > temp_pid.txt
for /f "tokens=2" %%a in (temp_pid.txt) do (
    echo PID: %%a
)
del temp_pid.txt 2>nul

Ventajas:
• Más legible y fácil de entender
• Funciona en todos los sistemas Windows
• Fácil de depurar (puedes ver temp_pid.txt si falla)
• Compatible con todos los idiomas
