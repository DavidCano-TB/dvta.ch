║                    SOLUCIÓN DE PROBLEMAS                                 ║
╚══════════════════════════════════════════════════════════════════════════╝

❌ "API key not found"
   → Verifica: cat config/.groq_key
   → Si no existe: echo "tu-api-key" > config/.groq_key

❌ "Invalid API key"
   → Ve a: https://console.anthropic.com/settings/keys
   → Crea una NUEVA key
   → Ejecuta de nuevo: CONFIGURAR_ANTHROPIC_API.bat

❌ "Insufficient credits"
   → Ve a: https://console.anthropic.com/settings/billing
   → Añade $20+ USD
   → Espera 1-2 minutos

❌ "Rate limit exceeded"
   → Espera 1 minuto
   → O añade más créditos

❌ La IA responde siempre "Ni sí ni no"
   → Ejecuta: python test_ai_simple.py
   → Mira los logs del servidor para ver el error
