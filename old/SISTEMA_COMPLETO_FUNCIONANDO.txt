SI EL SERVIDOR NO RESPONDE:
  1. Verificar que el proceso Python esté corriendo
  2. Verificar puerto 8000: netstat -ano | findstr :8000
  3. Revisar server.log para errores
  4. Reiniciar con: python _restart_all.py

SI NGROK NO FUNCIONA:
  1. Verificar que ngrok esté corriendo
  2. Verificar API: http://127.0.0.1:4040
  3. Revisar ngrok.log
  4. Reiniciar ngrok: ngrok http 8000

SI LAS APUESTAS NO FUNCIONAN:
  1. Verificar que el fetch use ${PORRA_ID}
  2. Abrir consola del navegador (F12)
  3. Verificar errores JavaScript
  4. Recargar con Ctrl+F5 (limpiar caché)

SI LAS VIDEOLLAMADAS NO FUNCIONAN:
  1. Verificar permisos de cámara/micrófono
  2. Abrir consola del navegador (F12)
  3. Ejecutar: window.videoDiagnostic()
  4. Verificar que WebSocket esté conectado
  5. Verificar ICE servers: /api/ice-servers
  6. Probar en navegador diferente (Chrome/Firefox)
