   └─ Inicia solo el servidor local
   └─ Modo interactivo (muestra logs)

10. ARRANQUE_AUTOMATICO_COMPLETO.bat
    └─ Inicia servidor + ngrok automáticamente
    └─ Modo background (sin consola)

11. ADMIN_INICIAR_NGROK.bat
    └─ Inicia ngrok con permisos de administrador
    └─ Verifica que el servidor esté corriendo primero

DIAGNÓSTICO:
12. VERIFICAR_SERVIDOR.bat
    └─ Verifica estado del servidor y ngrok
    └─ Muestra URLs y códigos HTTP

13. PRUEBA_COMPLETA.bat
    └─ Verifica que todo el sistema esté listo
    └─ Comprueba Python, dependencias, archivos, etc.

MANTENIMIENTO:
14. DETENER_TODO.bat
    └─ Detiene todos los servicios
    └─ Libera el puerto 8000

15. INSTALAR_DEPENDENCIAS.bat
    └─ Instala/actualiza dependencias Python
    └─ Verifica la instalación
