## 🔑 Puntos Clave

### 1. El Bote Total Incluye TODO
```
Bote Total = Dinero de TODAS las opciones
           = 200 (Equipo A) + 200 (Empate) + 600 (Equipo B)
           = 1000 DVDc
```

### 2. Solo los Acertantes Reciben Dinero
```
Acertantes = Solo quienes apostaron a "Equipo A"
           = Juan + María + Pedro
```

### 3. La Proporción se Calcula SOLO sobre la Opción Ganadora
```
Total a Opción Ganadora = 200 DVDc (solo "Equipo A")
Proporción de Juan = 100 / 200 = 50%
```

### 4. La Ganancia se Calcula sobre el Bote Total
```
Ganancia de Juan = 1000 × 50% = 500 DVDc
(Recibe el 50% del bote total de 1000 DVDc)
```

## 🎉 Conclusión

El sistema reparte el **100% del bote total** entre los acertantes, de forma **proporcional** a lo que cada uno apostó a la opción ganadora.

**Fórmula Simple:**
```
Tu Ganancia = Bote Total × (Tu Apuesta a Opción Ganadora / Total Apostado a Opción Ganadora)
```

**Ejemplo Rápido:**
- Bote: 1000 DVDc
- Apostaste: 100 DVDc a opción ganadora
- Total a esa opción: 200 DVDc
- Tu proporción: 100/200 = 50%
- **Tu ganancia: 1000 × 50% = 500 DVDc**
