## ❌ QUÉ NO DEBERÍA VER

Si ve alguno de estos problemas, el sistema NO está funcionando correctamente:

- ❌ Pantalla completamente en blanco
- ❌ Solo el header sin contenido
- ❌ Error "Conectando..." que no cambia a "Conectado"
- ❌ Error "Sin acceso OPO"
- ❌ Grid de botones vacío (sin números 1-30)
- ❌ Falta el título "⚙️ Configurar nueva partida"
- ❌ Falta el subtítulo "30 simulacros · 10 preguntas cada uno"

---

## 🔧 SOLUCIÓN DE PROBLEMAS

### Problema: Pantalla en blanco
**Solución**:
1. Abrir DevTools (F12)
2. Ir a Console
3. Buscar errores JavaScript
4. Verificar que el WebSocket está conectado (pestaña Network → WS)

### Problema: "Conectando..." no cambia
**Solución**:
1. Verificar que el servidor está corriendo: `netstat -ano | findstr ":8000"`
2. Verificar que el usuario tiene acceso OPO
3. Revisar logs del servidor

### Problema: No se puede hacer login
**Solución**:
1. Verificar credenciales
2. Usar password master si es necesario: `dvd_aGGDdCWQ5Bh3`
3. Verificar que el usuario no está bloqueado
