┌──────────────────────────────────────────────────────────────────────────────┐
│                              BACKEND API                                     │
└──────────────────────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────────┐
    │  main.py (FastAPI Server)                                   │
    │  ┌───────────────────────────────────────────────────────┐  │
    │  │  ROUTES                                               │  │
    │  │                                                        │  │
    │  │  GET /votaciones                                      │  │
    │  │  └─> Serves votaciones.html file                     │  │
    │  │                                                        │  │
    │  │  GET /api/votaciones/list                             │  │
    │  │  └─> Returns all votaciones for user                 │  │
    │  │      • Filters by user permissions                    │  │
    │  │      • Includes vote counts                           │  │
    │  │      • Marks user's votes                             │  │
    │  │                                                        │  │
    │  │  GET /api/votaciones/{votacion_id}                    │  │
    │  │  └─> Returns detailed info + statistics              │  │
    │  │      • All options with vote counts                   │  │
    │  │      • Percentages and progress                       │  │
    │  │      • Voter names (if DVD or not anonymous)          │  │
    │  │      • User's votes                                   │  │
    │  │                                                        │  │
    │  │  POST /api/votaciones/create                          │  │
    │  │  └─> Creates new votacion                            │  │
    │  │      • Validates title and options                    │  │
    │  │      • Stores in database                             │  │
    │  │      • Returns created votacion                       │  │
    │  │                                                        │  │
    │  │  POST /api/votaciones/votar                           │  │
    │  │  └─> Registers a vote                                 │  │
    │  │      • Validates votacion is open                     │  │
    │  │      • Checks if user already voted (if not multiple) │  │
    │  │      • Stores vote in database                        │  │
    │  │                                                        │  │
    │  │  POST /api/votaciones/finalizar                       │  │
    │  │  └─> Finalizes votacion and calculates results       │  │
    │  │      • Only creator or DVD can finalize               │  │
    │  │      • Counts votes per option                        │  │
    │  │      • Identifies winners (handles ties)              │  │
    │  │      • Creates complete ranking                       │  │
    │  │      • Stores results in database                     │  │
    │  │                                                        │  │
    │  │  DELETE /api/votaciones/{votacion_id}                 │  │
    │  │  └─> Deletes votacion completely                      │  │
    │  │      • Only creator or DVD can delete                 │  │
    │  │      • Removes all votes (CASCADE)                    │  │
    │  │                                                        │  │
    │  │  DELETE /api/votaciones/{votacion_id}/voto            │  │
    │  │  └─> Removes user's vote(s)                           │  │
    │  │      • Only if votacion is still open                 │  │
    │  │      • Removes all user's votes from that votacion    │  │
    │  └───────────────────────────────────────────────────────┘  │
    └─────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────────┐

    │  ┌───────────────────────────────────────────────────────┐  │
    │  │  Creates/Opens: data/apuestas.db                      │  │
    │  │                                                        │  │
    │  │  TABLE: votaciones                                    │  │
    │  │  ┌──────────────────────────────────────────────────┐ │  │
    │  │  │ id              INTEGER PRIMARY KEY              │ │  │
    │  │  │ creador         TEXT NOT NULL                    │ │  │
    │  │  │ titulo          TEXT NOT NULL                    │ │  │
    │  │  │ descripcion     TEXT                             │ │  │
    │  │  │ opciones_json   TEXT NOT NULL                    │ │  │
    │  │  │ multiple        INTEGER DEFAULT 0                │ │  │
    │  │  │ anonima         INTEGER DEFAULT 1                │ │  │
    │  │  │ estado          TEXT DEFAULT 'abierta'           │ │  │
    │  │  │ resultado_json  TEXT                             │ │  │
    │  │  │ created_at      TEXT DEFAULT CURRENT_TIMESTAMP   │ │  │
    │  │  │ finalizada_at   TEXT                             │ │  │
    │  │  └──────────────────────────────────────────────────┘ │  │
    │  │                                                        │  │
    │  │  TABLE: votos                                         │  │
    │  │  ┌──────────────────────────────────────────────────┐ │  │
    │  │  │ id              INTEGER PRIMARY KEY              │ │  │
    │  │  │ votacion_id     INTEGER NOT NULL                 │ │  │
    │  │  │ username        TEXT NOT NULL                    │ │  │
    │  │  │ opcion          TEXT NOT NULL                    │ │  │
    │  │  │ fecha           TEXT DEFAULT CURRENT_TIMESTAMP   │ │  │
    │  │  │ FOREIGN KEY (votacion_id) REFERENCES votaciones  │ │  │
    │  │  │   ON DELETE CASCADE                              │ │  │
    │  │  └──────────────────────────────────────────────────┘ │  │
    │  └───────────────────────────────────────────────────────┘  │

    │    ✅ See ALL vote amounts                                   │

    │    ✅ Can see percentages                                    │
    │    ✅ Can see progress bars                                  │
    │    ✅ Can vote                                                │
    │    ✅ Can remove own vote                                    │
    │    ✅ Creator can manage own votacion                        │
    │                                                              │
    │  ELSE (votacion.anonima == FALSE):                          │
    │    ✅ Can see voter names                                    │
    │    ✅ Can see all statistics                                 │
    │    ✅ Can vote                                                │
    │    ✅ Can remove own vote                                    │

┌──────────────────────────────────────────────────────────────────────────────┐
│                          SECURITY FEATURES                                   │
└──────────────────────────────────────────────────────────────────────────────┘

    ✅ JWT Token Authentication
       • All API calls require valid token
       • Token passed via Authorization header
       • Token extracted from localStorage

    ✅ Permission Checks
       • Only creator or DVD can finalize
       • Only creator or DVD can delete
       • Only authenticated users can vote

    ✅ Anonymity Protection
       • Voter names hidden if anonymous=true AND not DVD
       • Database stores all data, but API filters response
       • Frontend respects is_dvd flag

    ✅ Data Validation
       • Title required
       • Minimum 2 options
       • Cannot vote on finalized votacion
       • Cannot vote twice (unless multiple=true)

    ✅ Database Integrity
       • Foreign key constraints
       • CASCADE delete (removing votacion removes all votes)
       • Timestamps for audit trail
