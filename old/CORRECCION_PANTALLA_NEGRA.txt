PROBLEMA:
  La pantalla de juego se veia casi en negro
  El juego estaba ahi pero no se veia

CAUSA ENCONTRADA:
  Los modales (videoModal y victoryModal) tenian
  estilos CSS conflictivos en el HTML:
  
  display:none; display:flex;
  
  Esto causaba que los modales se quedaran
  visibles tapando la pantalla del juego

SOLUCION APLICADA:

1. MODAL DE VIDEO (videoModal)
   Antes: display:none;...;display:flex;
   Ahora: display:none;...;align-items:center;
   
   El display:flex se aplica via JavaScript
   cuando se necesita mostrar el modal

2. MODAL DE VICTORIA (victoryModal)
   Antes: display:none;...;display:flex;
   Ahora: display:none;...;align-items:center;
   
   El display:flex se aplica via JavaScript
   cuando se necesita mostrar el modal
