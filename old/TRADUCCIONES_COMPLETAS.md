## 🎨 SISTEMA DE TRADUCCIÓN

### Implementación
El sistema usa el atributo `data-i18n` en los elementos HTML:

```html
<!-- Texto del elemento -->
<button data-i18n="btnLogin">Sign in</button>

<!-- Placeholder de input -->
<input data-i18n-placeholder="placeholderUsername" />

<!-- Título de elemento -->
<div data-i18n-title="tooltipHelp"></div>
```

### Carga de Idiomas
```javascript
// Cargar idioma
await loadLang('es');  // Español
await loadLang('en');  // Inglés
await loadLang('fr');  // Francés
await loadLang('it');  // Italiano
await loadLang('ca');  // Catalán
await loadLang('de');  // Alemán
await loadLang('eu');  // Euskera

// Función de traducción con variables
t('admConfirmReset', { user: 'john' })
// → "Reset password for john? They will need to re-register."
```

### Aplicación Automática
```javascript
function applyTranslations() {
  // Traduce todos los elementos con data-i18n
  document.querySelectorAll('[data-i18n]').forEach(el => {
    const v = T[el.dataset.i18n];
    if (v !== undefined) el.innerHTML = v;
  });
  
  // Traduce placeholders
  document.querySelectorAll('[data-i18n-placeholder]').forEach(el => {
    const v = T[el.dataset.i18nPlaceholder];
    if (v !== undefined) el.placeholder = v;
  });
  
  // Traduce títulos
  document.querySelectorAll('[data-i18n-title]').forEach(el => {
    const v = T[el.dataset.i18nTitle];
    if (v !== undefined) el.title = v;
  });
}
```
