# Videos para Hundir la Flota

Este directorio contiene los videos que se reproducen durante el juego para hacer la experiencia más inmersiva.

## Videos necesarios:

### 1. **hit.mp4** - Impacto en barco
- Se reproduce cuando un jugador acierta un disparo
- Duración recomendada: 2-3 segundos
- Sugerencia: Explosión, fuego, impacto de misil

### 2. **miss.mp4** - Disparo fallido
- Se reproduce cuando un jugador falla el disparo (agua)
- Duración recomendada: 2-3 segundos
- Sugerencia: Salpicadura de agua, misil cayendo al mar

### 3. **sunk.mp4** - Barco hundido
- Se reproduce cuando un barco completo es hundido
- Duración recomendada: 3-5 segundos
- Sugerencia: Barco hundiéndose, explosión grande, barco bajo el agua

### 4. **win.mp4** - Victoria
- Se reproduce cuando un jugador gana la partida
- Duración recomendada: 3-5 segundos
- Sugerencia: Fuegos artificiales, celebración, victoria naval

### 5. **lose.mp4** - Derrota
- Se reproduce cuando un jugador pierde
- Duración recomendada: 3-5 segundos
- Sugerencia: Barco hundiéndose, derrota, pantalla oscura

### 6. **start.mp4** - Inicio de partida
- Se reproduce al comenzar una nueva partida
- Duración recomendada: 3-5 segundos
- Sugerencia: Barcos navegando, preparación para batalla

## Dónde conseguir los videos:

1. **Pixabay** (https://pixabay.com/videos/) - Videos gratuitos sin derechos de autor
2. **Pexels** (https://www.pexels.com/videos/) - Videos gratuitos de alta calidad
3. **Mixkit** (https://mixkit.co/) - Videos gratuitos para proyectos
4. **Videvo** (https://www.videvo.net/) - Videos gratuitos y de stock

## Búsquedas recomendadas:

- "missile explosion"
- "water splash"
- "ship sinking"
- "naval battle"
- "fireworks celebration"
- "ocean waves"
- "explosion fire"

## Formato recomendado:

- Formato: MP4 (H.264)
- Resolución: 1920x1080 (Full HD) o 1280x720 (HD)
- Tamaño: Menos de 5MB por video para carga rápida
- Sin audio o con audio de ambiente

## Nota:

Si no tienes videos disponibles, el juego funcionará perfectamente sin ellos. Los videos son opcionales y mejoran la experiencia visual, pero no son necesarios para la jugabilidad.
