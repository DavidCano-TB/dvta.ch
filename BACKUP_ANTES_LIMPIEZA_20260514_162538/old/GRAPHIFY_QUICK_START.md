# 🚀 Graphify - Guía Rápida

## ⚡ Instalación (1 minuto)

```bash
INSTALAR_GRAPHIFY.bat
```

## 📊 Generar Grafo (2-5 minutos)

```bash
GENERAR_GRAFO_CONOCIMIENTO.bat
```

## 📁 Resultados

Los archivos se generan en `graphify-out/`:

- **graph.html** → Visualización interactiva (abre en navegador)
- **GRAPH_REPORT.md** → Análisis del proyecto
- **graph.json** → Datos del grafo

## 🎯 ¿Para qué sirve?

✅ **Visualizar arquitectura** del proyecto  
✅ **Onboarding** de nuevos desarrolladores  
✅ **Documentación automática** del código  
✅ **Detectar dependencias** críticas  
✅ **Análisis de impacto** antes de refactorizar
