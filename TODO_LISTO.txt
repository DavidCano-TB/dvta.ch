🔧 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════════

Error: Python no encontrado
  → Instalar desde https://www.python.org/downloads/

Error: Dependencias faltantes
  → pip install -r requirements.txt
  → cd modules\exams && pip install -r requirements.txt

Error: Puerto ocupado
  → taskkill /F /IM python.exe

Error: Tunnel no funciona
  → Verificar cloudflared.exe --version
  → Revisar cloudflare-dvta-config.yml

Error: Sintaxis Python
  → python -m py_compile archivo.py
