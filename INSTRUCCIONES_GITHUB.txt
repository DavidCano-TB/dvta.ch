PASO 1: CREAR REPOSITORIO EN GITHUB
────────────────────────────────────

1. Ve a: https://github.com/new

2. Configuración del repositorio:
   • Repository name: dvdbank
   • Description: Sistema completo de gestión bancaria con juegos,
                  apuestas, votaciones y chat en tiempo real
   • Visibility: Private (recomendado) o Public
   • ❌ NO marques "Initialize this repository with a README"
   • ❌ NO agregues .gitignore (ya lo tienes)
   • ❌ NO agregues license (puedes hacerlo después)

3. Clic en [Create repository]

4. Copia la URL del repositorio que aparece
   Ejemplo: https://github.com/tu-usuario/dvdbank.git

PASO 2: CONECTAR REPOSITORIO LOCAL CON GITHUB
──────────────────────────────────────────────

Abre PowerShell en c:\dvdcoin\ y ejecuta:

# Agregar el remote de GitHub (reemplaza con tu URL)
git remote add origin https://github.com/TU-USUARIO/dvdbank.git

# Verificar que se agregó correctamente
git remote -v

# Deberías ver:
# origin  https://github.com/TU-USUARIO/dvdbank.git (fetch)
# origin  https://github.com/TU-USUARIO/dvdbank.git (push)

PASO 3: HACER PUSH AL REPOSITORIO
──────────────────────────────────

# Push de la rama master
git push -u origin master

# Si te pide autenticación:
# - Usuario: tu-usuario-github
# - Password: tu-token-personal (NO tu contraseña)

# Para crear un token personal:
# 1. Ve a: https://github.com/settings/tokens
# 2. Clic en [Generate new token] → [Generate new token (classic)]
# 3. Nombre: DVDBank Push Token
# 4. Expiration: 90 days (o el que prefieras)
# 5. Scopes: Marca "repo" (acceso completo a repositorios)
# 6. Clic en [Generate token]
# 7. Copia el token (solo se muestra una vez)
# 8. Usa ese token como contraseña al hacer push

Para no tener que ingresar usuario y token cada vez:

# Opción 1: Guardar credenciales en Windows Credential Manager
git config --global credential.helper wincred

# Opción 2: Guardar credenciales en cache por 1 hora
git config --global credential.helper cache

# Opción 3: Usar SSH en lugar de HTTPS (más seguro)
# 1. Generar clave SSH:
ssh-keygen -t ed25519 -C "davidcano.ch@gmail.com"

# 2. Agregar clave a GitHub:
#    https://github.com/settings/keys
#    Clic en [New SSH key]
#    Pega el contenido de: C:\Users\PC\.ssh\id_ed25519.pub

# 3. Cambiar remote a SSH:
git remote set-url origin git@github.com:TU-USUARIO/dvdbank.git
