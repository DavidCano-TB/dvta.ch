═══════════════════════════════════════════════════════════════════════════
  CÓMO AÑADIR NUEVA FUNCIONALIDAD
═══════════════════════════════════════════════════════════════════════════

Ejemplo: Añadir módulo de "Games" en puerto 8002

1. Crear módulo:
   C:\dvdcoin\modules\games\
   ├── start_games.py
   └── app_games.py

2. Abrir PowerShell como Administrador

3. Ejecutar:
   cd C:\dvdcoin\services
   
   .\add_new_service.ps1 `
       -ServiceName "DVDcoin-Games" `
       -DisplayName "DVDcoin Games Server" `
       -Description "Servidor de juegos (Puerto 8002)" `
       -WorkingDir "C:\dvdcoin\modules\games" `
       -Script "start_games.py" `
       -Port 8002

4. ¡Listo! El nuevo servicio está instalado y corriendo

═══════════════════════════════════════════════════════════════════════════
  COMANDOS ÚTILES
═══════════════════════════════════════════════════════════════════════════

PowerShell:

# Ver estado de todos los servicios
Get-Service -Name "DVDcoin-*"

# Iniciar todos
Start-Service -Name "DVDcoin-*"

# Detener todos
Stop-Service -Name "DVDcoin-*" -Force

# Reiniciar todos
Restart-Service -Name "DVDcoin-*"

# Ver logs recientes
Get-EventLog -LogName Application -Source "DVDcoin-*" -Newest 20

# Ver servicio específico
Get-Service -Name "DVDcoin-Bank" | Select-Object *
