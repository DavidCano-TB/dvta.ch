  ❓ PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════════════════

  Error 1033 en dvta.ch:
    → Ejecuta: ARREGLAR_DVTA_AHORA.bat

  Puerto ocupado:
    → Ejecuta: taskkill /F /IM python.exe
    → Luego: ACTIVAR_DVTA_CH_AHORA.bat

  dvta.ch no carga:
    → Espera 30-60 segundos
    → Recarga la página (Ctrl+F5)
    → Verifica estado: STATUS_DVTA.bat

  Dependencias faltantes:
    → cd modules\exams
    → pip install -r requirements.txt
