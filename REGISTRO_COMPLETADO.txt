═══════════════════════════════════════════════════════════════════
  🚀 CÓMO PROBAR (3 MINUTOS)
═══════════════════════════════════════════════════════════════════

1. INICIAR SERVIDOR
   > python src/main.py
   O usar: INICIAR_TODOS_SERVIDORES.bat

2. ABRIR NAVEGADOR
   → http://localhost:8000/bank

3. REGISTRAR USUARIO
   - Click en "Registrarse"
   - Completar formulario
   - Marcar "Estoy interesado en Oposiciones"
   - Aceptar términos
   - Click "Registrarse"

4. VERIFICAR EMAIL (Manualmente en desarrollo)
   > sqlite3 src/data/users.db
   > UPDATE users SET email_verified=1 WHERE username='testuser';
   > .quit

5. HACER LOGIN
   - Verificar que aparece pestaña "📚 Oposiciones"
   - Click para acceder

═══════════════════════════════════════════════════════════════════
  📧 ACTIVAR EMAIL (OPCIONAL)
═══════════════════════════════════════════════════════════════════

OPCIÓN 1: MailHog (Recomendado para desarrollo)
   1. Descargar: https://github.com/mailhog/MailHog/releases
   2. Ejecutar: mailhog
   3. Editar: config/email_config.json → "enabled": true
   4. Ver emails en: http://localhost:8025

OPCIÓN 2: Gmail
   1. Generar App Password: https://myaccount.google.com/apppasswords
   2. Editar config/email_config.json con tus credenciales
   3. Cambiar "enabled": true

═══════════════════════════════════════════════════════════════════
  🔍 VERIFICAR ESTADO
═══════════════════════════════════════════════════════════════════

Ver datos de usuario:
> sqlite3 src/data/users.db
> SELECT username, email, email_verified, opo_interest, 
  opo_access, payment_status FROM users;
> .quit

Activar acceso OPO manualmente:
> sqlite3 src/data/users.db
> UPDATE users SET opo_access=1, payment_status='completed' 
  WHERE username='testuser';
> .quit
